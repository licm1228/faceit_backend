# FaceIt 赛题对照表

本文档用于对齐“AI 模拟面试与能力提升平台”赛题要求与当前仓库实现，便于答辩、材料整理和自检。

## 1. 岗位化题库与知识库

已实现：

- 支持 `Java后端`、`Web前端开发`、`Python算法` 三类岗位。
- 初始化数据包含约 `120` 道题，覆盖技术题、项目题、场景题、行为题等题型。
- 题目支持岗位、难度、关键词标签、参考答案管理。
- 岗位接口已显式返回：
  - `requiredSkills`
  - `interviewFocus`
  - `evaluationWeights`
- 推荐练习已关联：
  - 题型
  - 关键词标签
  - 推荐原因
  - 岗位知识来源说明

对应实现：

- 后端：`/position/*`、`/question/*`
- 评估权重：`PositionProfileService`
- 推荐练习：`InterviewChatService.buildRecommendedPractices`

## 2. 多模态交互式模拟面试

已实现：

- 文本输入面试。
- 语音录制 + 语音转文字。
- 动态追问，多轮对话单题最多追问 3 次。
- 聊天式面试与经典面试页两条流程都支持面试启动与报告查看。

补充说明：

- 语音识别由 `/speech/recognize/base64` 提供。
- 浏览器不支持后端录音时，会自动降级为浏览器语音识别。

对应实现：

- 前端：`InterviewPage.tsx`、`ChatInput.tsx`
- 后端：`InterviewController`、`InterviewChatController`

## 3. 面试表现多维度分析

已实现：

- 内容分析维度：
  - 技术准确性
  - 岗位匹配度
  - 逻辑清晰度
  - 知识覆盖度
- 表达分析维度：
  - 语速
  - 停顿占比
  - 清晰度
  - 自信度
  - 流畅度
  - 表达总分
- 面试报告支持结构化输出：
  - 总分
  - 五个维度分数
  - 表达摘要
  - 亮点
  - 不足
  - 改进建议
  - 推荐练习

对应实现：

- 核心评估：`AIEvaluationService`
- 报告生成：`InterviewChatService.buildBaseReport`、`InterviewReportPage.tsx`

## 4. 能力提升反馈闭环

已实现：

- 面试历史记录。
- 成长曲线。
- 用户能力档案。
- 弱项标签识别。
- 学习建议列表。
- 个性化推荐练习。

当前闭环路径：

1. 完成面试
2. 生成结构化报告
3. 聚合弱项标签
4. 返回学习建议与推荐练习
5. 前端能力档案区展示“下一步”和建议题目

对应实现：

- 后端：`UserProfileService`
- 前端：`InterviewPage.tsx`

## 5. 当前答辩建议

建议演示顺序：

1. 先选不同岗位，展示岗位画像与评分权重差异。
2. 进入语音作答，展示表达分析卡片。
3. 完成面试后进入报告页，展示五维评分和推荐练习。
4. 返回能力档案区，展示弱项标签、学习建议和下一练题目。

建议强调点：

- 不是通用问答，而是岗位化模拟面试。
- 不只给内容分，还给表达分析。
- 不只生成报告，还形成“练习-评估-提升”闭环。
